# Unlock website

## Setup

At first, you must have [pnpm](https://pnpm.io/).

To install, run:

```bash
curl -fsSL https://get.pnpm.io/install.sh | sh -
```

or follow other the instructions [here](https://pnpm.io/installation).

## Installation

1. `pnpm i`
2. `pnpm start`
3. A browser window will open up, pointing to the docs.
4. Profit!
